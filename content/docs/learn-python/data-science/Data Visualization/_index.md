### ✅ Data Visualization (Matplotlib & Seaborn)
- Plotting line, bar, scatter plots
- Statistical visualization with Seaborn

**Example:**
```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]
plt.plot(x, y)
plt.show()
```
