### ✅ Data Structures
- List methods (`append()`, `pop()`, slicing)
- Dictionaries (`keys()`, `values()`)
- Sets (`union()`, `intersection()`)

**Example:**
```python
scores = [10, 20, 30]
scores_set = set(scores)
print(scores)
```
