## 🚀 Level 1: Python Fundamentals

### ✅ Basic Syntax & Data Types
- Variables and data types (`int`, `float`, `str`, `bool`, `list`, `tuple`, `set`, `dict`)
- Arithmetic operations (`+`, `-`, `*`, `/`, `//`, `%`, `**`)
- Input and Output (`input()`, `print()`)

**Example:**
```python
a = 10
b = 5
print(f"Sum: {a + b}")
```
